#Altering tables
Altering -> done to add, modify of delete data in a table
delete example - alter table <tName> drop COLUMN <colName>;
adding example - alter table <tName> add COLUMN <colName>;
#Addition of a new primary key
-alter table <tableName> add constraint <Cname> primary key <colName>
#Modification of datatype
alter table <tName> MODIFY COLUMN <colName> <newDataType>;
